# Урок 2: Основные операции с комплексными числами

Теперь, когда каждое комплексное число стало точкой на плоскости, пришло время научиться ими управлять.  
Мы разберём основные операции: равенство, сложение, вычитание, умножение, деление, а также познакомимся с такими важными понятиями, как модуль и сопряжённые числа.

---

### 1. Равенство комплексных чисел

Два комплексных числа <MathExpression inline>{"z_1 = a + bi"}</MathExpression> и <MathExpression inline>{"z_2 = c + di"}</MathExpression> равны тогда и только тогда, когда равны их действительные и мнимые части.

<MathExpression displayMode>{"a = c \\quad \\text{и} \\quad b = d"}</MathExpression>

Нельзя сказать, что одно комплексное число «больше» или «меньше» другого. На плоскости точки нельзя выстроить в один ряд, поэтому для комплексных чисел операторы <MathExpression inline>{">"}</MathExpression> и <MathExpression inline>{"<"}</MathExpression> **не имеют смысла**.

---

### 2. Сложение и вычитание

Комплексные числа складываются и вычитаются **по частям** — отдельно действительная, отдельно мнимая.

<MathExpression displayMode>{"(a + bi) \\pm (c + di) = (a \\pm c) + (b \\pm d)i"}</MathExpression>

**Пример сложения:**

<MathExpression displayMode>{"(3 + 2i) + (1 - 4i) = (3 + 1) + (2 - 4)i = 4 - 2i"}</MathExpression>

Геометрически это обычное сложение векторов по правилу параллелограмма:

  points={[
    { re: 3, im: 2, label: "3 + 2i", color: "#3b82f6" },
    { re: 1, im: -4, label: "1 - 4i", color: "#10b981" },
    { re: 4, im: -2, label: "4 - 2i", color: "#ef4444" }
  ]}
  vectors={[
    { from: {re:0, im:0}, to: {re:3, im:2}, color: "#3b82f6" },
    { from: {re:3, im:2}, to: {re:4, im:-2}, color: "#10b981" },
    { from: {re:0, im:0}, to: {re:4, im:-2}, color: "#ef4444" }
  ]}
/>

Вычитание <MathExpression inline>{"z_1 - z_2"}</MathExpression> геометрически означает прибавление к первому вектору противоположного <MathExpression inline>{"(-z_2)"}</MathExpression>.

---

### 3. Модуль комплексного числа

**Модуль** комплексного числа <MathExpression inline>{"z = a + bi"}</MathExpression> (обозначается как <MathExpression inline>{"|z|"}</MathExpression>) — это расстояние от начала координат до точки <MathExpression inline>{"z"}</MathExpression> на комплексной плоскости.

По теореме Пифагора длина вектора равна:

<MathExpression displayMode>{"|z| = \\sqrt{a^2 + b^2}"}</MathExpression>

**Пример:**  
Если <MathExpression inline>{"z = 3 - 4i"}</MathExpression>, то его модуль:
<MathExpression displayMode>{"|z| = \\sqrt{3^2 + (-4)^2} = \\sqrt{9 + 16} = \\sqrt{25} = 5"}</MathExpression>

В отличие от самих комплексных чисел, модули — это обычные действительные неотрицательные числа, и их **можно сравнивать** между собой (кто дальше от нуля).

---

### 4. Комплексно-сопряжённые числа

Если у комплексного числа изменить знак мнимой части на противоположный, мы получим **сопряжённое** число. Оно обозначается чертой сверху:

<MathExpression displayMode>{"z = a + bi \\quad \\Rightarrow \\quad \\overline{z} = a - bi"}</MathExpression>

Геометрически это зеркальное отражение точки относительно действительной оси (оси X).

<StaticComplexPlane
  points={[
    { re: -2, im: 3, label: "-2 + 3i", color: "#3b82f6" },
    { re: -2, im: -3, label: "-2 - 3i", color: "#ef4444" }
  ]}
/>

**(Замечательное свойство!)**  
Произведение числа на его сопряжённое всегда даёт квадрат модуля:

<MathExpression displayMode>{"z \\cdot \\overline{z} = (a+bi)(a-bi) = a^2 - (bi)^2 = a^2 - b^2(-1) = a^2 + b^2 = |z|^2"}</MathExpression>

Это свойство поможет нам научиться делить комплексные числа.

---

### 5. Умножение

Умножение выполняется как обычное раскрытие скобок, с заменой <MathExpression inline>{"i^2"}</MathExpression> на <MathExpression inline>{"-1"}</MathExpression>:

<MathExpression displayMode>{"(a + bi)(c + di) = ac + adi + bci + bdi^2 = (ac - bd) + (ad + bc)i"}</MathExpression>

**Пример:**

<MathExpression displayMode>{"(2 + 3i)(1 - 4i) = 2 - 8i + 3i - 12i^2 = 2 - 5i - 12(-1) = 2 - 5i + 12 = 14 - 5i"}</MathExpression>

**Геометрический смысл (самое важное!):**

Умножение комплексного числа на другое число — это одновременно:
- Поворот вектора на угол аргумента второго числа
- Растяжение (или сжатие) вектора пропорционально модулю второго числа

Особенно красиво работает умножение на мнимую единицу:
<MathExpression displayMode>{"z \\cdot i"}</MathExpression> — поворачивает вектор <MathExpression inline>{"z"}</MathExpression> ровно на <MathExpression inline>{"90^\\circ"}</MathExpression> против часовой стрелки, не меняя его длину.

    { re: 0, im: -1, label: "-i", color: "#f59e0b" }

  vectors={[

    { from: {re:0, im:0}, to: {re:0, im:-1}, color: "#f59e0b" }
  ]}
/>

### 6. Деление

Как разделить одно число на другое, если в знаменателе стоит <MathExpression inline>{"i"}</MathExpression>? Нам нужно от него избавиться!  
Для этого мы умножаем и числитель, и знаменатель дроби на **сопряжённое** знаменателю число. Тогда в знаменателе получится обычное действительное число (квадрат модуля).

<MathExpression displayMode>{"\\frac{z_1}{z_2} = \\frac{a + bi}{c + di} = \\frac{(a + bi)(c - di)}{(c + di)(c - di)} = \\frac{ac - adi + bci - bdi^2}{c^2 + d^2}"}</MathExpression>

**Пример:**

Разделим <MathExpression inline>{"1 + 5i"}</MathExpression> на <MathExpression inline>{"1 + 2i"}</MathExpression>.
Умножаем обе части на сопряжённое знаменателя: <MathExpression inline>{"1 - 2i"}</MathExpression>.

<MathExpression displayMode>{"\\frac{1 + 5i}{1 + 2i} = \\frac{(1 + 5i)(1 - 2i)}{(1 + 2i)(1 - 2i)} = \\frac{1 - 2i + 5i - 10i^2}{1^2 + 2^2}"}</MathExpression>

<MathExpression displayMode>{"= \\frac{1 + 3i + 10}{1 + 4} = \\frac{11 + 3i}{5} = \\frac{11}{5} + \\frac{3}{5}i = 2.2 + 0.6i"}</MathExpression>

Таким образом мы «очищаем» результат от мнимой части в знаменателе и получаем ответ в стандартной алгебраической форме <MathExpression inline>{"a + bi"}</MathExpression>.

---

### Итог урока

Комплексные числа подчиняются обычным алгебраическим правилам с единственной оговоркой, что <MathExpression inline>{"i^2 = -1"}</MathExpression>:
- **Модуль** <MathExpression inline>{"|z|"}</MathExpression> — это длина вектора на плоскости.
- **Сопряжённое число** <MathExpression inline>{"\\overline{z}"}</MathExpression> помогает избавиться от <MathExpression inline>{"i"}</MathExpression> при умножении и делении, так как <MathExpression inline>{"z \\cdot \\overline{z} = |z|^2"}</MathExpression>.
- Сложение работает как параллельный перенос векторов, а умножение — как поворот и масштабирование.
- Деление сводится к искусственному переносу мнимой части из знаменателя в числитель с помощью сопряжения.

Данные свойства делают комплексные числа незаменимым инструментом в физике, инженерии и компьютерной графике.

**В следующем уроке** мы научимся использовать алгебраическую форму для решения любых уравнений (в том числе находить корни из отрицательных чисел).
